Message formats

For all
8 bytes - Client Long UID
8 bytes - Component Type Long UID
1 byte - Bit-flags.
    - Nothing (256)
    - Nothing (128)
    - Nothing (64)
    - Nothing (32)
    - Nothing (16)
    - Nothing (8)
    - IsRemoved (4)
    - IsEvent (2)
    
For not removed component
-> For all
8 bytes - Network Entity Long UID
2 bytes - Size
{size} bytes - Component Bytes

For removed component
-> For all
8 bytes - NetworkEntity Long UID

For events
-> For all
2 bytes - Size
{size} bytes - Event bytes
